# Middleman3::Sassc

This gem lets you use SassC with Middleman 3 by monkey patching relevant sass tilt templates.
(Middleman 4 supports SassC out of box.)

## Installation

In `Gemfile`:
```ruby
gem 'middleman3-sassc', '~> 0.0.1'
```

In `config.rb`:

```ruby
activate :sassc
```
